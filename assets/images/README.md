# Asset Folder

## Images Folder
Place your photos here and name them as:
- photo1.jpg
- photo2.jpg
- photo3.jpg
- photo4.jpg
- photo5.jpg
- photo6.jpg

Recommended format: JPG or PNG
Recommended size: 800x800 pixels (square format)

## CPR Certificate
Place your CPR certificate PDF in the parent assets folder (not in images):
- cpr-certificate.pdf

The website will automatically link to these files once you add them.
