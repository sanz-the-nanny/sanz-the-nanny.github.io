## Setup Instructions

### Adding Your Photos
1. Place your photos in the `assets/images/` folder
2. Name them as: `photo1.jpg`, `photo2.jpg`, `photo3.jpg`, `photo4.jpg`, `photo5.jpg`, `photo6.jpg`
3. Recommended image size: 800x800 pixels (square format)

### Adding Your CPR Certificate
1. Place your CPR certificate PDF in the `assets/` folder
2. Name it: `cpr-certificate.pdf`

### Customizing Testimonials
Edit the `index.html` file in the testimonials section to add real testimonials from your clients.
